# DIY Guide

Welcome to the Zeroday build guide.

This document explains the baseline path for building a project with a Raspberry Pi Zero 2 W.

## Suggested flow

1. Pick one clear project goal.
2. Break it into hardware and software milestones.
3. Keep a journal with daily commits.
4. Share progress early and iterate.

## Build rhythm

- Day 1 to Day 2: plan and sketch
- Day 3 to Day 7: prototype core functionality
- Day 8 to Day 12: polish and test
- Day 13 to Day 14: document and publish

## Deliverables

- A public repository
- Clear setup instructions
- Photos or renders of your build
- A short demo video
